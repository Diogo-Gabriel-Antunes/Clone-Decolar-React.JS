# Como utilizar

1.Faça download dos arquivos

2.Tenha instalado o node.js na sua maquina

3.Tenha tambem algum gerenciador de pacote como por exemplo: npm ou yarn

4.Faça download das dependencias 

exemplo: 
<h3>npm install</h3>

5.Apos isso rode o projeto 

exemplo: 
<h3>npm run start</h3>
